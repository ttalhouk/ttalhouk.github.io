Basic Methods

is_a?(type)
This is useful for checking the type of object somthing is. Handy when iterating nested arrays/hashes.
to_s(or i, f, etc...)
Used to change object from one type to another such as "5" to 5.
split(pattern)
Used to split up strings. Placing (//) as the pattern divides each character up and puts it in an array.
reverse(!)
Reverses the order of a string or array. Using (!) modifies the object permenently.

Arrays

map(!)
Iterates through an array, creating a new array or modifying the existing one if (!) is used.
each(_index)
iterates through an array without modifying it. Can select the content or the index.
join(connector)
joins all elements with the connector in between.

Hashes

keys - values
Creates an array of the Keys or Values of the hash.
each_pair
Iterates through a hash with access to the key/value pair.
select
Selects key value pairs based on criteria given.

Loops, and Flow Control

if...else
Checks the condition of the if statement. If true it will run the block otherwise runs the esle block if given.
while (condition)
This will repeat the block of code until the condition becomes false.
&& || !
&& represents "and" so both statments must be true for it to be true. || is "or" and will evaluate to be true if one statement is true. ! is "not" and will evaluate opposite of the statement.
