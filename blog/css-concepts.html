Blog Entry Week 3: CSS

Not Just Boring Text Anymore

February 25, 2016

This week we've added styling to our site using CSS. So now you are saying "that's great, but what does that mean?" Well CSS stands for Cascading Style Sheets and controls the look and layout of an HTML page. It takes all those tags and gives them some type of styling like p{font-size:72px;} will turn something like <p>text</p> into...

text

Pretty cool huh! Well that's not all. Each element of a page has an invisible border around (though you can use CSS to show the border!) to keep parts of a page from running into each other but you may be asking how can I control the space that this border takes up, well CSS can help with that too through margin & padding. Let's take a look at this diagram:

CSS is also used to position items on the page and tell the page how other blocks of information should interact with them, like that CSS image at the top. By using img{float:left;} it told the page to push the image to the left and have all the text wrap around it. Pretty useful to make things flow nicely.

You can also move blocks around from their default positions by using the you guessed it... position property. Positions can be relitive using position: relitive followed by top: (N)px meaning move the top of this object N pixels from where it normally would be. Alternatively, using position: absolute followed by top: (N)px tells the page to place this N pixels from the top of the page no matter what else is there!

All this can get pretty confusing if your page becomes fairly complicated with many boxes and elements, so I've found that using the the Developer Tools on Google Chrome helps you visualize what is acting on what so you can pick the right tool to make you site look the way you want.

There are many other topics to cover on CSS that I can't possibly cover them all in one post. But if you want to add it there's probably a way to do it. Any tricks you've found. Send me an email and let me know. Thanks for reading!
