Blog Entry Week 2: HTML

The language of the internet!

This week we began to code in HTML. But even before that we had to understand the structure of a website.

In order to make a website developers commonly go through 6 stages

Info Gathering:
What is the purpose of the site? What is the goal of the site? Who is the target audience?
Planning:
This is the sage which the developer will create a site map to organize how the site will link together.
Design:
What is the site supposed to feel like? Colors, fonts, images should be determined to tell a cohesive story and enhance UX.
Development:
Development of the Home Page is done and the shell for all interior pages.
Testing & Delivery:
How does the site appear on diferent browesers? Is the site accessible for page readers? Mobile Devices?
Maintenance:
Periodic mainenance of the site is needed to make sure all links and funcitonality is working as updates to HTML and browsers can lead to issues if not maintained.

During the week we developed site maps and wireframes for our own sites. This laid the foundation in setting the code for the site development

Once the site was properly layed out, time to add the code! HTML5 allows for symantic tags to be used. These create blocks within the page that are laid out logicly and the source code can be read easily. This helps when developing, formatting and fixing bugs in the site. These also improve accessiblity and UX. These tags include <header>, <footer>, <nav>, and many more!

Personal Reflections:

While I've had some experience in the past with HTML and Page Making applictions, getting into the nuts and bolts of HTML5 was a fulfilling experience. It was interesting to learn about the development considerations of building a site and the different theories on what colors to use, what fonts make good headings or text for readablity and accessibility. So far the sites are not took much to look at but next week we will get into CSS. This allow us to turn the HTML into what we have visioned in our wireframes!
